## Community Science Museom

A responsive website created for the Community Science Museum.  
The project was built as part of a front-end development assignment and focuses on creating an engaging and accessible user experience for families and children interested in science.

The site provides information about the museum, exhibitions, events, and visitor details while maintaining a visually engaging design with strong colors, images, and interactive elements.

## Features

- Responsive layout for desktop and mobile
- Informational pages about exhibitions, events, and museum visits
- Image gallery showcasing museum experiences
- Accessible HTML structure with semantic elements and alt text
- Styled call-to-action buttons with visual effects

## Prerequisites

No special prerequisites are required.  
You only need a modern web browser.

Examples:
- Chrome
- Firefox
- Edge
- Safari

## Getting Started

### Installation

Clone the repository:

git clone https://github.com/moaisason/pm-project.git

### Running the project

Open the project folder and launch index.html in your browser.

```bash
open index.html
```
or simply double-click the file.
